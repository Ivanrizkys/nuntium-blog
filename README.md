## Stacks
- NextJs
- TailwindCS

## Setup in Local

### Preprequisites
- `node`, minimum version 16.14.0
- `yarn`, [see installation instruction](https://classic.yarnpkg.com/lang/en/docs/install/)

### Development
- Install all dependencies, by running `yarn`
- Run in local, using command `yarn dev`
